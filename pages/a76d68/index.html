# 在 Vue 项目中使用 axios 的三种方式

首先 npm i axios, npm 下载 axios 插件.

# 第一种方式:

直接在 vue 组件中导入 axios, 并直接引用.

注意一点, axios 是一个基于 promise (opens new window) 网络请求库, 这意味着, 你必须使用 ES6 的 then() 或者 ES8 的 async/await 来处理 Promise 实例对象. (各位, 我这里用的是 async/await.)

components/Hello.vue:
// 导入 axios
import axios from 'axios';
1
2
3
// 方法
methods: {
        // async/await 
        async initCartList() {
            // 解构赋值          // 发起 axios 请求
            const { data:res } = await axios({
                method: 'GET',
                url: 'https://www.xxxx.cn/api/cart',
            });
            // console.log(res);
            if (res.status === 200) return  this.data = res.list;
        },
}

# 背景音乐

今天给大家分享几个网页背景音乐代码，非常简单适用，在网页制作中经常要用到的。

一，HTML5标签的video：

<video controls="" autoplay="" name="media"><source src="音乐" type="audio/mpeg"></video>
1

这种方式会显示播放器。

二，embed标签

<embed src="music/We Don't Talk Anymore.mp3" autostart="true" loop="true" hidden="true"></embed>
1

这个方式主要是支持的播放器多。一定程度上解决了浏览器兼容的问题。

三，audio标签

<audio autoplay="" loop=""><source src="http://www.d163.net/Letter_Song.mp3"></audio>
1

这个不显示播放器。

四，bgsound

<bgsound src=背景音乐链接地址 loop=-1>
1

五，audio

<audio src="music/We Don't Talk Anymore.mp3" id="andyguo" autoplay="autoplay" controls="controls" preload="auto"></audio>
1

以上就是网页背景音乐代码的全部内容。
